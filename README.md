
Скопировать
```
git clone https://github.com/justmedic/TokkariTailor
```

Войти в директорию
```
cd Tokkaritailor
```

Докер
(миграции и тд делать ниче не надо)
```
docker-compose up --build
```

Сайт
```
http://127.0.0.1:8001/
```

Админка:
admin 12345678
(есть скрипт авторегистрации админа для тестов)
```
http://127.0.0.1:8001/admin/
```
